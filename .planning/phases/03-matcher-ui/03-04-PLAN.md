    ## Change 3 — Add refreshData() function

    After the existing `loadData` function, add the following new function:

    ```js
    async function refreshData() {
      refreshing.value = true
      error.value = null
      try {
        const res = await fetch('/api/matches/refresh', { method: 'POST' })
        if (!res.ok) throw new Error(`Refresh feilet: ${res.status} ${res.statusText}`)
        // Re-fetch all data after successful refresh
        await loadData()
      } catch (e) {
        error.value = e.message
      } finally {
        refreshing.value = false
      }
    }
    ```

    Note: `loadData()` already sets `loading.value = true` internally, so the user will see
    the loading spinner during the data reload phase after a successful refresh.

    ## Change 4 — Add toolbar with "Oppdater data" button to template
